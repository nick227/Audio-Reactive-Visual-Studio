# Abstract / Art Nouveau / Esoteric FX Pack

This pack adds a new concept collection focused on abstract, symbolic, bizarre, experimental, and eclectic art direction.

## Collection

- **Art Nouveau / Esoteric**

## Design intent

The goal is to move beyond standard DJ visuals into stranger, more authored territory:

- art-nouveau curves
- gilded ceremonial tones
- occult and symbolic motifs
- bizarre object language
- clever cross-style collisions
- editorial + mystical hybrids

## Pack contents

The pack introduces new assets across existing runtime families:

- gradients
- textures
- particles
- shapes
- frames
- typography
- visualizers
- 3D hero objects
- motion effects
- cutouts

## Why it fits the architecture

No new renderer types were required. The pack is data-first and stays inside the existing template system, so future packs can be added quickly with the same pattern.
